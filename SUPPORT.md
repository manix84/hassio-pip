# Support 🙋

HA TV PiP is early-stage software.

## Getting Help 🧭

For now, use GitHub issues for:

- Android TV build problems.
- PiP behavior bugs.
- Test stream playback problems.
- Documentation improvements.

Please include:

- Device or emulator model.
- Android version.
- App version or commit.
- Steps to reproduce.
- Relevant logs, if available.

## Not Supported Yet 🚧

These features are planned for later and are not supported in Phase 1:

- Home Assistant integration.
- Camera feeds.
- Device pairing.
- Discovery.
- WebSocket control.
- WebRTC.
- Remote access.
