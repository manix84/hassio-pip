# AI Declaration 🤖

HA TV PiP is a human-led personal project.

## Project Ownership 👤

The project is owned, directed, and maintained by Rob.

Architecture decisions, product direction, implementation choices, testing decisions, release decisions, code authorship, and final code ownership remain human responsibilities. AI tools do not own, maintain, write, or independently author project code.

## How AI Is Used 🛠️

AI tools may be used as an assistant for:

- Documentation drafts and wording cleanup.
- Release notes and changelog preparation.
- Planning notes, roadmap cleanup, and issue triage.
- Pull request and code review support.
- Test ideas and quality-check reminders.
- Repetitive project maintenance tasks.

This is intended to reduce admin overhead and keep the boring project work moving.

## How AI Is Not Used 🚫

AI assistance should not be interpreted as replacing the human maintainer or writing project code.

AI is not presented as the project author, coder, sole coder, or independent maintainer. Project code is written, reviewed, and accepted by the human maintainer. Any AI-assisted suggestion must still be reviewed, accepted, changed, or rejected by the human maintainer before it becomes part of the project.

## Accountability ✅

All committed changes are the responsibility of the human maintainer.

If something is wrong, unclear, unsafe, badly designed, or poorly documented, treat that as a project issue to report or improve rather than as an AI-authorship distinction.
