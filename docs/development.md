# Development

## Android TV App

Open `android-tv-app/` in Android Studio. The project uses Gradle Kotlin DSL, Kotlin, Compose, and Media3.

Command-line build:

```sh
cd android-tv-app
./gradlew assembleDebug
```

## Home Assistant Integration

The Home Assistant custom integration is intentionally not implemented in Phase 1. Placeholder files live under `ha-integration/custom_components/ha_tv_pip/`.
