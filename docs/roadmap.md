# Roadmap

## Phase 1: Android TV PiP MVP

- Android TV Kotlin app.
- Public test HLS playback.
- Media3 ExoPlayer.
- Manual PiP entry.
- Automatic PiP entry when Home is pressed.
- Basic logging and error handling.

## Future Phases

- Local HTTP control endpoint.
- mDNS discovery.
- Device pairing.
- Home Assistant config flow.
- `ha_tv_pip.show_camera` Home Assistant service.
- Home Assistant camera HLS stream support.
- Snapshot support.
- WebRTC support.
