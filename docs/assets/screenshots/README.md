## Privacy Notes 🔐

- Do not commit screenshots that show real Home Assistant URLs, bearer tokens, pairing codes, IP addresses, device IDs, personal profiles, or private camera feeds.
- Overlay screenshots captured over a real TV launcher should be reviewed carefully because launcher profiles, app rows, recommendations, and media titles may be personal.
- Prefer demo cameras, public test streams, or neutral generated backgrounds for future app-store and website screenshots.
