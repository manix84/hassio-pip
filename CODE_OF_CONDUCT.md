# Code of Conduct 🤝

This project aims to be practical, respectful, and useful.

## Expected Behavior ✅

- Be kind and direct.
- Assume good intent.
- Keep feedback specific and actionable.
- Respect different skill levels and home lab setups.
- Help keep issues and discussions focused on HA TV PiP.

## Unacceptable Behavior 🚫

- Harassment, threats, or personal attacks.
- Discriminatory language or behavior.
- Repeated off-topic comments.
- Sharing private information without permission.
- Abusive or intentionally disruptive participation.

## Reporting 📨

If you see behavior that does not fit this Code of Conduct, open a private security or maintainer contact path if one is available, or contact the project maintainer directly.

Maintainers may remove comments, close issues, or block participants when needed to keep the project healthy.
